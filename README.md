# weCode2015

Seven Layers is bridging technology professionals through partnering and creating diversity in tech one pair at a time.

# To run the app

Open your CLI and navigate to the project folder.

1. type `npm install` in the CLI (if you haven't already installed)

2. type `node app.js` to start the server

3. Visit `http://localhost:8889` in your browser

4. To stop serving the page to your browser, press Control+C in the CLI.
